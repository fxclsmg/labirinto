# labirinto

O que é:
Este projeto tem como objetivo emular a criação de labirintos de forma procedural, bem como emular personagens capazes de encontrar uma saída.

Funcionalidades:
Criar labirinto de forma procedural com S saídas
Criar C caminhos de tamanho T a partir de uma saída com até V curvas
Posicionar P personagens em um caminho
Personagem tenta encontrar saída do labirinto
